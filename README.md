# Olist Ecommerce Data Analytics

A personal data analytics project analyzing 2 years of Brazilian ecommerce data.

## What I Built
- Python pipeline that downloads and processes 99,000+ orders weekly
- SQL Server database with star schema (6 tables)
- Power BI dashboard with 4 pages of analysis

## Tools Used
- Python 3.12
- SQL Server 2025
- Power BI Desktop

## What I Found
- Olist grew 100x in 14 months
- Delivery time dropped from 54 days to 7 days
- São Paulo accounts for 60% of all orders
- Beauty & Health is the top revenue category

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Add your database settings to `.env`
3. Run the pipeline: `python main.py`
